Beginner Simple Calculator

Build a functional calculator using HTML, CSS, and JavaScript

Project Instructions & Milestones:
  1. HTML: Create a display area for the calculator output and buttons for digits (0-9), decimal, operators (+, −, ×, ÷), clear, and equals.
  2. CSS: Style the calculator to have a modern look with distinct buttons and a large display.
  3. JavaScript:
    • Implement button clicks to display numbers and operators in the display area.
    • Calculate the result when equals is pressed.
    • Implement clear button to reset the calculator display.
    • Handle basic error cases, such as dividing by zero.
  4. Bonus: Add keyboard support for number and operator inputs.

Follow these milestones inside the respective parts to complete the calculator project!
